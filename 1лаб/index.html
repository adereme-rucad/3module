Лабораторная работа 1

Счёт: 0

Управление змейкой: wasd, кнопки - стрелки

Задания:

  1. Добавьте обработчики нажатия на клавиши стрелок.
  2. Добавьте возможность управления змейкой при помощи кнопок. Для этого добавьте на игровое поле простейший джойстик со стрелками.
  3. При нажатии на кнопки змейка должна реагировать также как и при нажатии на клавиши. (Менять направление движения).
  4. Добавьте возможность сброса игры по нажатию на клавишу R.
  5. Добавьте отображение текущего количества набранных очков.
  6. Добавьте пользовательский элемент ввода, позволяющий задать скорость
