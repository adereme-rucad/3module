Лабораторная работа 1 - змейка

Управление змейкой: wasd, кнопки - стрелки

R - рестарт

Задания:

  1. Добавьте обработчики нажатия на клавиши стрелок.
  2. Добавьте возможность управления змейкой при помощи кнопок. Для этого добавьте на игровое поле простейший джойстик со стрелками.
  3. Добавьте возможность сброса игры по нажатию на клавишу R.
  4. Добавьте отображение текущего количества набранных очков.
  5. Добавьте пользовательский элемент ввода, позволяющий задать скорость
